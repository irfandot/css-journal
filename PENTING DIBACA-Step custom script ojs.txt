{cara meletakan script CSS/Html pada menu right sidebar:}
*login sbg journal manager 
*klik menu journal manager
*klik system plugin
*klik generic plugin--klik enable jika belum, jika suda klik setting custom block maanager--buat nama block plugin baru kemudian save.
*klik system plugin--klik custom block manager--cari block baru yg dibuat tadi--edit, kemudian klik icon (html)dibawah dan paste script sesuaikan dengan kebutuhan 
NB: Keterangan *<a href* --> adalah informasi link yg mengarah ke URL tujuan
    Keterangan * Img Src*--> adalah informasi sumber gambar yg diambil dari sumbernya
